# Screenshots de Dydi

Esta carpeta está destinada a hospedar la evidencia visual de Dydi, ideal para mostrar flujos y el pulido de la UI a reclutadores o profesores sin requerir que clonen o configuren Supabase localmente.

## Capturas Pendientes

Actualmente requerimos que alguien con las credenciales activas del `.env` local levante el proyecto en su máquina (`docker compose up`) y tome capturas de las siguientes vistas clave. 

*(Por favor, recorta las imágenes de forma limpia, usando bordes o sombras ligeras si es posible, y optimízalas en formato WebP o PNG liviano)*:

1. **`01_login.png`**: Vista de la pantalla de inicio de sesión de la web, mostrando el diseño y la marca.
2. **`02_squad_dashboard.png`**: Vista principal del dashboard de hábitos donde se listan los miembros del squad y el estado de sus check-ins del día.
3. **`03_proposals.png`**: La lista de hábitos propuestos donde el equipo bota (interfaz de comunidad/acuerdos).
4. **`04_roulette.png`**: (Opcional pero recomendado). Una captura del momento en que se activa la ruleta de penitencias o el resultado final de la misma.

Una vez tomadas, guárdalas en este directorio y actualiza los enlaces en la galería del `README.md` en la raíz.
